#### Usage

How to install, instantiate, or embed the fonts elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/scripts/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var fonts = Patterns.fonts();
    </script>

##### Module Import

    import Fonts from 'src/elements/fonts/fonts';

    new Fonts();
